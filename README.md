
## Project Description

This FTOptix sample project demonstrates how to implement **Locale Retentivity on Application Restart without Multisessions**. It provides an example of persisting the user's selected language/locale preference across application restarts in single-session HMI environments, ensuring the interface language is restored when the application is relaunched.
